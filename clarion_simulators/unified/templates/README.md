# templates/

Python source templates loaded at runtime by `execution_tracer.pl` via `read_template/2`.

These were extracted from inline string literals to keep the Prolog module readable.

## Files

- **analyze_paths.py** — Execution path analysis script (PyMC Bayesian inference, Stan via CmdStanPy, frequentist MLE). Generated by `pgm_to_python_package/3`.
- **gnn_vae.py** — GNN-VAE (Graph Neural Network Variational Autoencoder) for learning latent representations of execution traces. Uses PyTorch Geometric. Generated by `generate_gnn_vae_package/2`.
